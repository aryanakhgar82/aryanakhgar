# Black Friday Quiz — وبسایت مسابقه‌ای
این پروژه یک قالب کامل برای مسابقه پرسش‌و‌پاسخ (Quiz) با طراحی مدرن و افکت شیشه‌ای است. مشخصات اصلی:
- 2000 سوال (فایل `questions.json`) — دسته‌بندی‌شده به سری‌های ۵تایی
- تایمر دایره‌ای برای هر سوال
- دکمه نئونی «بزن بریم» با قابلیت روشن/خاموش
- افکت گلریزون (confetti) زمانی که شرکت‌کننده ۵ پاسخ درست در یک دسته بدهد
- پس‌زمینه شیشه‌ای چندلایه، تصاویر وسایل برقی محو در پیش‌زمینه
- آماده برای آپلود در GitHub — شامل فایل ZIP

## جایگزینی تصاویر و فونت‌ها
- پوشه `assets/` شامل فایل‌های نمونه با نام `appliance-1.jpg`, `appliance-2.jpg`, `appliance-3.jpg`.
  این‌ها را با تصاویر دارای لایسنس مناسب (Unsplash, Pexels یا تصاویر خریداری‌شده) جایگزین کنید.
- فونت پیش‌فرض: Vazirmatn (Google Fonts). در صورت نیاز فونت محلی خود را آپلود کرده و در `index.html` ارجاع دهید.

## راه‌اندازی محلی
1. از حالا می‌توانید پروژه را در مرورگر باز کنید: `index.html`
2. اگر با خطا در بارگذاری `questions.json` مواجه شدید، مطمئن شوید که فایل در همان پوشه باشد.

## ساختار پروژه
- index.html
- style.css
- app.js
- questions.json
- assets/ (تصاویر نمونه)
